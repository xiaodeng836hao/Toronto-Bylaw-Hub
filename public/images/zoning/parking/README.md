# Parking guide infographics

The `/zoning/parking` page (`app/zoning/parking/ParkingClient.tsx`) references three
infographics by these EXACT file names. Save the three provided "Parking Requirements"
images here with these names (PNG):

| File name                          | Image (panel title)                         | Shown in section                |
| ---------------------------------- | ------------------------------------------- | ------------------------------- |
| `yard-location-guide.png`          | Residential Zone – Yard Location Guide      | "Where parking may be located"  |
| `commercial-vehicle-guide.png`     | Residential Zone – Commercial Vehicle Guide | "Commercial Parking"            |
| `recreational-vehicle-guide.png`   | Residential Zone – Recreational Vehicle Guide | "Recreational Vehicle Parking" |

Each reference cites Zoning By-law 569-2013 · 10.5.80.10 and is shown as an
"Informational reference only" illustration. Intrinsic size is set to 1040×1480 in
the data file (`data/zoning/parking-topics.ts`); the actual images can be any size with
a similar portrait aspect ratio.
